# AI + Crypto News — Frontend

Next.js 16 (App Router) frontend for the AI + Crypto News aggregator.
See the [root README](../README.md) for the full project overview.

```bash
npm install
npm run dev        # http://localhost:3000
```

Reads articles from `../data/processed-articles.json` (kept up to date by the
scheduled pipeline in the repo root).
